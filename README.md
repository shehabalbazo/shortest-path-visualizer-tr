# 🧠 Graph Algorithms Visualizer

An interactive web-based tool that visualizes graph algorithms step by step using HTML, CSS, JavaScript, and vis.js.

The project demonstrates how classical graph algorithms work in real time with animations, logs, and dynamic updates.

---

## ⚡ Live Demo

🔗 **Try it here:**  
https://shehabalbazo.github.io/shortest-path-visualizer/

---

## 🚀 Features

- 📊 Interactive graph visualization using vis-network
- 🧭 Dijkstra Algorithm (Shortest Path)
- 🔁 Bellman-Ford Algorithm (supports negative weights)
- 🌳 Prim Algorithm (Minimum Spanning Tree)
- 🌲 Kruskal Algorithm (Minimum Spanning Tree)
- ⏯️ Step-by-step animation (Play / Pause / Next / Back)
- 📋 Real-time distance table updates
- 📝 Detailed execution logs
- 🎨 Clean and modern UI design

## 🧠 Algorithms

### 🔵 Dijkstra
Finds the shortest path from a starting node to all other nodes using a greedy approach (no negative weights).

### 🔴 Bellman-Ford
Computes shortest paths from a single source and supports negative weights.

### 🌳 Prim
Builds a Minimum Spanning Tree by expanding from a starting node using minimum edges.

### 🌲 Kruskal
Builds a Minimum Spanning Tree by sorting all edges and using Union-Find to avoid cycles.
